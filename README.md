# Fabric toImageData

## Getting Started
To use the library
### 1. Import
```javascript
require('fabric-to-image-data')('fabric');
```
or
```javascript
import fabric from 'fabric'
import FabricToImageData from 'fabric-to-image-data'
FabricToImageData(fabric)
```
### 2. Call function
```javascript
fabricCanvasInstance.toImageData()
fabricObjectInstance.toImageData()
```
